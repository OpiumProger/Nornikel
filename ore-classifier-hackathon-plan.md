## Стратегия: гибрид CV + сегментация + экспертные правила

Жюри оценит не только F1, а **геологическую логику, интерпретируемость и интеграцию в лабораторный процесс**. Поэтому ядро:

```
TIFF/PNG → тайлинг → предобработка → сегментация (4 класса) → постобработка → метрики → правила → отчёт/UI
```

### 4 класса маски (обязательно)

| Класс | Цвет   | Смысл              |
|-------|--------|--------------------|
| 0     | —      | Фон (матрица)      |
| 1     | 🟢 зелёный | Обычные срастания  |
| 2     | 🔴 красный | Тонкие срастания   |
| 3     | 🔵 синий   | Тальк              |

### Финальная классификация руды — не нейросеть, а правила из ТЗ

```python
if talc_pct > 10:
    ore_class = "оталькованная"
elif ordinary_pct > fine_pct:
    ore_class = "рядовая"
else:
    ore_class = "труднообогатимая"
```

## Как выжать качество при малых данных

1. **Предаудит разметки** (2–3 часа вместе): одинаковые ли цвета талька, нет ли путаницы обычные/тонкие.
2. **Старт с классики для сульфидов**, если разметки мало:
   - Otsu / adaptive threshold на L-канале LAB → светлые сульфиды;
   - доля «серой фазы внутри контура» → эвристика обычное vs тонкое:

   ```python
   replacement_ratio = dark_pixels_inside / sulfide_area
   if replacement_ratio < 0.25:
       class = "ordinary"
   else:
       class = "fine"
   ```

   Это **fallback** и объяснение жюри: «геологическая морфология + ML для талька».

3. **Тальк — приоритет для нейросети** (размечен цветной линией, ±3% критично).
4. **Валидация**: 80/20 по **образцам**, не по патчам (иначе утечка).
5. **Ансамбль не делать** — нет времени.
